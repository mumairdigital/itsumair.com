# Microcopy and UI Strings

Every small piece of text in the interface. Prepared here so development never has to invent copy.

## Buttons
| Context | Text |
|---|---|
| Primary CTA | Get My Free Audit |
| Primary CTA (alt) | Get My Free Visibility Audit |
| Booking | Book a 20-Minute Review |
| Secondary | See Results |
| Service card | See How It Works |
| Case study card | Read the Full Story |
| Blog card | Read More |
| Form submit | Send My Request |
| Phone | Call Now |
| Nav CTA | Free Audit |

Never use: Submit, Click Here, Learn More (alone), Get Started (too vague).

## Form messages
| State | Message |
|---|---|
| Required field | Please fill this in |
| Invalid email | That email doesn't look right |
| Invalid phone | Please check that number |
| Submitting | Sending... |
| Success | Got it. I'll get back to you within one business day. |
| Error | Something went wrong. Email me directly at [email] and I'll sort it out. |

## Trust microcopy
- Under CTAs: "Free. No obligation. 24-hour turnaround."
- Near forms: "I'll never share your information."
- Response promise: "I reply within one business day."
- Contract: "No long-term contracts."
- Ownership: "You own everything I build."
