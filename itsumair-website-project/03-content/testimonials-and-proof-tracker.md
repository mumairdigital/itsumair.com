There are no Tampa bathroom-remodeler case studies at launch. This is the single biggest credibility gap in the project.

**Absolute rule: never fabricate, imply, or exaggerate proof.** One invented testimonial discovered destroys the entire "truth over theatre" positioning and is unrecoverable.

## Three honest approaches to the proof gap

**Approach A — Adjacent work, clearly labeled**
Present real tile/showroom and technical work as "related work," never as bathroom-remodeler marketing results. Honest and immediately available.

**Approach B — First Tampa client at reduced rate**
Trade a discount for a documented, publishable case study with full data access. Highest long-term value. Make this the top business priority.

**Approach C — Public teardown**
Publicly analyze an anonymized Tampa bathroom remodeler's visibility problems. Demonstrates skill without claiming a client. Doubles as blog content and outreach material.

**Recommendation: A + C at launch, B as the immediate priority.**

## Testimonial request process

Ask existing clients with a specific prompt so they do not stall:

> "Could you write 3–4 sentences covering: what the problem was before we worked together, what I did, and what changed? Specific numbers help if you have them."

Collect: full name, company, city, photo, permission to publish.

## Testimonial usage rules
- Never anonymous ("A happy client") — reads as fake
- Always full name + company
- Photo wherever possible
- Never edit the meaning; light trimming only
- If you have none, omit the section entirely rather than inventing one
