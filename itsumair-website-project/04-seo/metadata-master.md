## Rules
- Brand name only where character count allows
- Pipe separator
- No keyword stuffing, no ALL CAPS, no excessive punctuation
- Meta descriptions written as ad copy with an implied CTA
- Every page gets Open Graph and Twitter card tags

## Open Graph requirements
Per page: `og:title`, `og:description`, `og:image` (1200x630), `og:url`, `og:type`
Twitter: `summary_large_image`

Test rendering on LinkedIn and Facebook before launch.
