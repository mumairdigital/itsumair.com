# Pre-Launch Checklist

Nothing goes live until every box is ticked. Grouped by discipline.

## Content
- [ ] Every page has final approved copy — no placeholder text anywhere
- [ ] No lorem ipsum in any file
- [ ] No fabricated testimonials, results, client names, or logos
- [ ] Every statistic and claim is verifiable
- [ ] Reading level checked (target: 8th grade)
- [ ] No em dashes (house style)
- [ ] Spelling set to US English throughout
- [ ] Business name, email, phone identical on every page (NAP consistency)
- [ ] Legal pages complete: privacy, terms, accessibility statement
- [ ] "Not based in Tampa" question answered honestly on the site

## Technical
- [ ] HTTPS active, valid certificate
- [ ] One canonical host enforced (www or non-www), other 301s
- [ ] Trailing slash convention consistent site-wide
- [ ] Self-referencing canonical on every page
- [ ] XML sitemap generated and accessible
- [ ] robots.txt correct and not blocking anything important
- [ ] Custom 404 page live with navigation
- [ ] No broken internal or external links
- [ ] Redirect map applied and tested, no chains
- [ ] Favicon and app icons set
- [ ] Core Web Vitals green on mobile: LCP < 2.5s, INP < 200ms, CLS < 0.1
- [ ] Tested on real mobile hardware, not just emulator
- [ ] Tested in Chrome, Safari, Firefox, Edge
- [ ] Tested on iOS Safari specifically (most common failure point)

## SEO
- [ ] Unique title tag on every page, under 60 characters
- [ ] Unique meta description on every page, under 155 characters
- [ ] Exactly one H1 per page
- [ ] Logical heading hierarchy, no skipped levels
- [ ] Alt text on every meaningful image
- [ ] Internal linking implemented per map
- [ ] No orphan pages — everything within 3 clicks of home
- [ ] All schema validates in Google Rich Results Test
- [ ] Open Graph and Twitter card tags on every page
- [ ] Google Search Console verified and sitemap submitted
- [ ] GA4 installed and firing
- [ ] Conversion events configured and tested

## Accessibility
- [ ] WCAG 2.1 AA contrast verified — especially all #6700C8 usage
- [ ] Full keyboard navigation works
- [ ] Visible focus states on all interactive elements
- [ ] Form labels correctly associated
- [ ] Skip-to-content link present
- [ ] Tested with a screen reader
- [ ] Body text minimum 16px
- [ ] Tap targets minimum 44x44px

## Conversion
- [ ] One clear primary CTA on every page
- [ ] Phone number click-to-call on mobile
- [ ] Contact form tested end-to-end, email received
- [ ] Audit form tested end-to-end
- [ ] Calendar booking tested, including from a US timezone
- [ ] Confirmation messages and emails working
- [ ] Spam protection active and not blocking real submissions
- [ ] Sticky mobile CTA bar working
- [ ] Thank-you pages configured for conversion tracking

## Security and operations
- [ ] Backups configured and a restore tested
- [ ] SPF, DKIM, DMARC configured for the sending domain
- [ ] Admin access secured with 2FA
- [ ] Domain registrar lock enabled
- [ ] Domain auto-renew enabled
- [ ] Dependencies up to date, no known vulnerabilities
- [ ] Rate limiting or spam protection on forms
- [ ] Uptime monitoring configured

## Brand consistency
- [ ] Headshot matches all social profiles
- [ ] Same handle referenced across all platforms
- [ ] Same email used everywhere
- [ ] Social links correct and live
- [ ] OG share image renders correctly (test on LinkedIn and Facebook)

## Final review
- [ ] Read every page on a phone, outdoors, in daylight
- [ ] Ask a non-marketer to explain what the site offers after 10 seconds
- [ ] Confirm nothing on the site targets homeowners
