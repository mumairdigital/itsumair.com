# Post-Launch Plan

## First 14 days — monitoring
- Check Google Search Console daily for crawl errors and indexation
- Verify pages are being indexed (site: search and URL Inspection)
- Monitor form submissions — send yourself test submissions weekly
- Watch Core Web Vitals field data as real traffic arrives
- Fix anything broken immediately; do not batch fixes in this window

## Ongoing cadence

| Frequency | Task |
|---|---|
| Weekly | Publish or promote content; check form submissions; review GSC coverage |
| Bi-weekly | Review analytics: which pages convert, which do not |
| Monthly | Technical health check (speed, broken links, uptime); update case studies with new results |
| Quarterly | Content refresh of top pages; review keyword positions; competitor check |
| Annually | Full audit; design refresh assessment; domain and licensing renewals |

## Content pipeline after launch
Priority order for pages built after launch:
1. Additional case studies as clients are signed — the single highest-value addition
2. Blog posts per approved cadence
3. Location pages 7–15 as local proof accumulates
4. Expanded FAQ based on real questions prospects ask

## Measurement
Track these, in this priority:
1. Audit form submissions
2. Booked calls
3. Signed clients
4. Assisted conversions (which pages appear in converting journeys)
5. Keyword positions for buyer-intent terms
6. Sessions (lowest priority — see project-brief.md)

## Feedback loop
Every sales call is research. After each call, log:
- What objection came up that the site did not answer
- What question they asked that should be in the FAQ
- What proof they asked for that does not exist yet

Feed these back into the site monthly. The site should get sharper with every conversation.

## When to revisit strategy
- If after 6 months no clients have been signed, review the acquisition channel before blaming the site
- If Tampa proves saturated, the expansion path is Sarasota/Bradenton then Fort Myers (see prior metro analysis)
- If bathroom-only proves too narrow to fill capacity, widen to kitchen and bath — but only after the bathroom position is established
